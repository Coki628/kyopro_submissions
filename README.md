# kyopro_submissions

## Prerequisites
1. Python 3.6
2. pip
3. pipenv

## Setup

```bash
# setup virtual env
pipenv --python 3.6
pipenv shell
# install dependencies
pip install -r requirements.txt
```
- change your python path accordingly in .vscode/settings.json
- change your python path accordingly in .vscode/tasks.json

```bash
# create dist files
mkdir _dist
touch _dist/main.cpp
```
